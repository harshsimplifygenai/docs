Verbeo is an AI voice agent platform that lets you build and deploy intelligent phone agents in minutes. Whether you need to handle inbound support calls, run large-scale outbound campaigns, or automate routine phone interactions, Verbeo gives you a simple REST API and dashboard to configure, launch, and monitor agents at scale — without managing telephony infrastructure.

## How it works

Every interaction on Verbeo is built around three core resources: **agents**, **phone numbers**, and **calls**.

An agent is the AI entity you configure — it holds the system prompt, voice, tools, and call settings. A phone number is the caller ID your agent uses when placing outbound calls or receiving inbound calls. A call is a single phone conversation, with a full transcript, duration, status, and optional analysis data attached after it ends.

You manage all three through the Verbeo dashboard or directly through the REST API at `https://api.verbeo.ai`.
