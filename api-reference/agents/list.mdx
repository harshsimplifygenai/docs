This endpoint returns every agent in the workspace associated with your API key. No query parameters are required — the workspace is automatically inferred from your key. Use the [Get agent](/api-reference/agents/get) endpoint to retrieve a single agent by ID.

## Endpoint

```
GET https://api.verbeo.ai/agents
```

## Request

This endpoint takes no query parameters and no request body.

<CodeGroup>

```bash cURL
curl -X GET "https://api.verbeo.ai/agents" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```javascript Node.js
const response = await fetch("https://api.verbeo.ai/agents", {
  headers: {
    Authorization: "Bearer YOUR_API_KEY",
  },
});
const agents = await response.json();
```

```python Python
import requests

response = requests.get(
    "https://api.verbeo.ai/agents",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
)
agents = response.json()
```

</CodeGroup>

## Response

A successful request returns `200 OK` with an array of agent objects.

<ResponseField name="[].id" type="string" required>
  UUID of the agent.
</ResponseField>

<ResponseField name="[].workspace_id" type="string" required>
  UUID of the workspace that owns this agent.
</ResponseField>

<ResponseField name="[].agent_name" type="string" required>
  Display name of the agent.
</ResponseField>

<ResponseField name="[].created_at" type="string" required>
  ISO 8601 timestamp of when the agent was created.
</ResponseField>

<ResponseField name="[].updated_at" type="string" required>
  ISO 8601 timestamp of the last update.
</ResponseField>

<ResponseField name="[].config" type="object" required>
  The full configuration object for the agent, including system prompt, voice, tools, and settings.
</ResponseField>

<ResponseField name="[].sip_uri" type="string">
  Full SIP URI for the agent. Present only when SIP was enabled for this agent.
</ResponseField>

<ResponseField name="[].twilio_sip_username" type="string">
  SIP username. Present only when SIP was enabled.
</ResponseField>

<ResponseField name="[].twilio_sip_domain" type="string">
  SIP domain. Present only when SIP was enabled.
</ResponseField>
